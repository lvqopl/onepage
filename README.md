## ✨ 功能特性

### 🔐 权限管理
- **双模式访问**：访客模式（只读）+ 管理员模式（完全控制）
- **密码保护**：管理功能需要登录后才能使用
- **会话保持**：登录状态自动保存

### 📚 书签管理
- **添加/编辑/删除**：完整的书签 CRUD 操作
- **自动图标**：自动获取网站 Favicon
- **自定义图标**：支持使用自定义图标 URL
- **URL 验证**：智能识别和验证网址

### 🏷️ 分类系统
- **动态分类**：自由创建和删除分类
- **拖拽分类**：拖动书签到分类标签快速更改分类
- **智能过滤**：按分类筛选显示书签
- **批量操作**：支持批量修改分类和删除

### 🎨 界面设计
- **现代 UI**：采用渐变背景和毛玻璃效果
- **响应式布局**：完美适配桌面、平板、手机
- **暗色模式**：可设置自定义背景图片
- **动画效果**：流畅的过渡和悬停效果

### 🔄 导入导出
- **JSON 导出**：导出所有书签数据
- **JSON 导入**：批量导入书签（自动去重）
- **HTML 导入**：支持导入浏览器导出的书签文件
- **智能去重**：自动识别并跳过重复书签

### 🔍 搜索功能
- **多引擎支持**：Google、Bing、百度、DuckDuckGo
- **快捷搜索**：Enter 键快速搜索
- **搜索建议**：实时显示当前搜索引擎

### ⚡ 其他特性
- **拖拽排序**：自由调整书签顺序（需登录）
- **多选操作**：支持批量选择和操作
- **即时保存**：所有更改自动保存
- **操作提示**：友好的成功/错误提示
